CSS 用来干什么？
</STitle>

前文提到过，CSS 是用来指定文档如何展示给用户的一门语言——如网页的样式、布局、等等。

一份**文档**是由标记语言组织起来的文本文件 —— [<u>HTML</u>](https://developer.mozilla.org/zh-CN/docs/Glossary/HTML) 是最常见的标记语言，但你可能也听说过其他可标记语言，如 [<u>SVG</u>](https://developer.mozilla.org/zh-CN/docs/Glossary/SVG) 或 [<u>XML</u>](https://developer.mozilla.org/zh-CN/docs/Glossary/XML)。
